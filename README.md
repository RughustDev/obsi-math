# obsi-math

Interactive mathematical graphing plugin for Obsidian built with TypeScript and WebGL.

## Usage

### Function graphing

Write a function inside an `obsi-math` code block:

````
```obsi-math
f(x)=x^2
```
````

Obsi Math will render the expression and display an interactive graph directly inside your note.

### Systems of equations

Write a system inside an `obs-sistema` code block:

````
```obs-sistema
x+y=5
2x-y=1
```
````

Obsi Math will solve the system and display the corresponding mathematical information.

> Note: This project is experimental and currently available only in Spanish.
